# Nexus — Condensed Description

Nexus is the human coding agent — a video call with a photorealistic avatar that builds, runs, researches, and explains software in real time. The thesis: the most natural UI a human ever invented is another human face, and now that AI writes code in seconds, the bottleneck has shifted from creation to understanding. Speak an idea and a Cursor SDK agent (composer-2) writes it into a Daytona sandbox booted in under 90 ms; the running app loads in an iframe beside the avatar that built it. Ask for advice and Exa returns a synthesized one-paragraph answer pulled live from the web; ask for an explanation and the avatar searches fal.ai's catalog, picks the right model at runtime — image, video, audio, 3D — and drops the generated visual into the panel. Gemini 3.1 Flash Live drives one realtime model that emits audio for Tavus Phoenix-4 (BYO-LLM, lip-synced 1080p at 40 fps) and structured tool calls for the orchestrator on a single socket, with sub-300 ms interruption. Convex streams every agent event reactively to the React UI; LiveKit Agents handles WebRTC. Two architectural bets shape it: **one Gemini for both voice and intent** (rejecting the cheaper two-model topology to keep a single transcript and get native interruption for free), and **Cursor talks to Daytona directly over MCP, cloud-to-cloud** — our orchestrator forwards events but never executes generated code, so the data path is sandboxed end-to-end.
